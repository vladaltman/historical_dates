# Интерактивная временная шкала

Это одностраничное React-приложение, отображающее окружность временной шкалы с анимацией и переключаемыми секциями разных событий определенных годов. Реализовано с использованием **React**, **TypeScript**, **GSAP**, **Swiper**, **Webpack**, **Sass**.

## Запуск проекта в режиме разработки

```bash
npm start
```

## Установка зависимостей

```bash
npm install
```

## Сборка проекта (production)

```bash
npm run build
```
Результат будет в папке dist/

## Структура проекта

├── node_modules                # Модули
├── public/                     # HTML-шаблон
│   └── index.html
├── src/                        # Исходники React-приложения
│   ├── data/                   # Тексты для слайдеров
│   ├── fonts/                  # Шрифты
│   ├── Styles/                 # Стили SCSS для компонентов
│   ├── App.tsx                 # Корневой компонент
│   ├── CircleComponent.tsx     # Компонент-окружность
│   ├── Content.tsx             # Блок, содержащий основной контент приложения. Независимый, многократный.
│   ├── index.tsx               # Точка входа приложения
│   └── SliderComponent.tsx     # Компонент-слайдер временных событий
├── .babelrc                    # Конфигурация для Babel
├── package.json                # Скрипты и зависимости
├── package-lock.json           # Точные версии всех установленных пакетов и их зависимостей
├── README.md                   # Файл документации
├── tsconfig.json               # Конфигурация TypeScript
└── webpack.config.js           # Конфигурация Webpack
